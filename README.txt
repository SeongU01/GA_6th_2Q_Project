# 프로젝트 개요
| 항목            | 내용                                                                 |
|-----------------|----------------------------------------------------------------------|
| **프로젝트**    | Direct2D 자체엔진을 활용한 게임 제작                                  |
| **장르**        | 실시간 전략 초스피드 덱빌딩 카드게임                                  |
| **진행 방식**   | 시간을 자원으로 사용하는 실시간 전략 게임                                            |
| **게임 설명**   | 플레이어는 최초의 강화 인간 ‘제로’를 플레이하여, 제한 시간 동안 모든 적을 처치해야만 한다. |
